# Mobile-First Interface Development

## Directives
- **Viewport Constraints:** Tailor all user forms (especially time-selection inputs, checkboxes, and avatar upload elements) for small mobile touchscreens. Ensure buttons are easily tappable.
- **Responsive Layouts:** Use Tailwind CSS responsive utilities (`sm:`, `md:`, `lg:`) to ensure components scale beautifully from an iPhone screen up to a manager's desktop dashboard monitor.
